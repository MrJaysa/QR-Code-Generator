QR-Code Generator

Hello there, This is an open source project developed with Python Programming Language using a Flask, which is a web microframework for building web applications with Python. This project focuses on creating QR-Codes for any link entered and providing a downloadable QR-Code as image files in PNG format, get an svg equivalent in the later updates and also for test purposes this app can be integrated to any applications requiring it's services.

QR-Code can be generated for any link directly on this website or you can use the API provided to generate QR-Code for your apps.

You can check out this project on the github repository link here. more updates to this app is coming soon.

Generate your QR-Code on the fly, just input the url and hit enter

You can also send a post request via ajax, fetch or any http request you are familiar with as shown below.

// i'll be using javascript fetch request, ensure your form has a named field 'url'.

let formData = new FormData('your-form')

fetch('{{ url_for('create_qr', _external=True) }}', {

method: 'post',

body: formData

}).then(res=>res.json()

).then(

res=>{console.log(res.data)}

)
